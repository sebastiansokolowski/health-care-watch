# mHealth Guard
![image](https://github.com/sebastiansokolowski/mHealth-Guard/blob/master/mobile/src/main/res/mipmap-xxxhdpi/ic_launcher.png)

Mobile Health Guard is an app to monitor your health in the real time using sensors from a smartwatch. 
The app analyze recaived data to predict:
- hearth rate anomally
- fall
- epilepsy attack

When health care event has been found the app can:
- send a SMS message to selected contacts
- notify user by show android notification

You can easly start and stop measurement when you want. Every algorithm can be configured for individual needs.
